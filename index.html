Pour tous niveaux : lycée, cpge, post-bac

Agrégé, polytechnicien, expérimenté, j’accompagne les élèves de tous niveaux en mathématiques. Mon objectif : redonner confiance, consolider les bases, préparer aux examens et aux concours.

Les maths méritent mieux que le sort qui leur est réservé par les programmes aujourd'hui.

Faire des maths, ce n'est pas remplir des QCM, appliquer des recettes, agiter une calculatrice surpuissante.

Faire des maths, c'est : comprendre avant d'apprendre ; s'approprier pour maîtriser ; raisonner pour convaincre ; chercher pour résoudre échouer avant de réussir ; conceptualiser et imaginer pour progresser. C'est dans cet esprit que j'enseigne les mathématiques.
